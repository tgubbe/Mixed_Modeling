This project was part of my Master's program (Data Analysis methods). The goal of the project was to explore four mixed effects models 
(2 linear and 2 nonlinear) and compare best model fit based on AIC values. Next, best linear unbiased predictors (BLUPs) for both the
fixed and random effects were calculated for individual patients. This would assist doctors in evaluating changed in CD4 cell count
trajectories over various treatment periods. In addition to the PDF report and R code, I provided the data file and the four .png images
of the model results.
